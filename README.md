# Filament v5 Ecommerce Kit

[![Lint & Test PR](https://github.com/achyutkneupane/ecommerce-kit/actions/workflows/tests.yml/badge.svg)](https://github.com/achyutkneupane/ecommerce-kit/actions/workflows/tests.yml)

This is a starter kit for building ecommerce using Filament v5 as admin panel.

## Installation

You can create a new Laravel project using this starter kit by running the following command:

```shell
laravel new --using=achyutn/ecommerce-kit
```

You will be asked to enter the name of your project. After that, the installation will proceed, and you will have a new Laravel project set up with Filament.
